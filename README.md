# Prototip - PySide6 Application

Este es un prototipo de aplicación construida con PySide6 que utiliza un diseño moderno tipo "Flat Dark".

## Características
- Interfaz gráfica moderna con PySide6.
- Tema oscuro optimizado.
- Estructura modular para fácil expansión.

## Requisitos
- Python 3.x
- PySide6

## Instalación
```bash
pip install -r requirements.txt
```

## Ejecución
```bash
python main.py
```
